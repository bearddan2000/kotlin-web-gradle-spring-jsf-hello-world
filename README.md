# kotlin-web-gradle-spring-jsf-hello-world

## Description
A springboot secure web app with jsf support.

## Tech stack
- kotlin
- gradle
  - springboot
  - jsf
  - bootstrap
  - jquery
  - sockjs

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`
Available at http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
